# インストーラー化

現在はZIP配布。将来的にセットアップウィザード形式にする場合のメモ。

## 概要

- スタートメニューへの登録
- デスクトップショートカットの自動作成
- 「アプリと機能」からのアンインストール対応

## 実装方法

### Inno Setup（推奨）
- Windows向け無料インストーラー作成ツール
- GitHub Actionsに組み込み可能
- `.iss` スクリプトを書いてビルドするだけ

### MSIX（Windowsストア対応）
- Windows 10/11のモダンな配布形式
- Flutter公式でMSIXパッケージ対応あり（`msix` パッケージ）
- ストア外配布も可能（サイドロード）

## 注意点

- コード署名なしだと初回起動時に「発行元不明」警告が出る
- 署名証明書の取得にはコストがかかる（年数万円〜）
- 社内配布のみなら警告を無視してもらうか、グループポリシーで回避できる
