UP-001: DOINGセクションの視覚的優先度強化
============================================

## 課題
全4セクション（DOING / STOCK / REVIEW / DONE）が同じ視覚的重みで
表示されており、最重要セクションであるDOINGが埋もれている。
ペルソナは「今日やること」を一目で把握したい。

## 影響
- PC狭幅（300px）：スクロールなしでDOINGが目に入るが、他と差がない
- モバイル：画面タップ直後に「今何をやるべきか」が瞬時に判断できない

## 対応内容
1. DOINGセクションの背景色を他セクションより明るく（#1E293B → #1a2844）
2. DOINGヘッダーのフォントサイズを 11px → 13px に拡大
3. DOINGセクションに微細な左ボーダー幅を 3px → 4px に変更
4. DOINGが空でない場合、セクション背景に薄いアクセントカラーのグラデーション

## 変更ファイル
- lib/widgets/kanban_section.dart

## 確認観点
- PC 300px幅で、DOINGが最も目立つこと
- モバイルで、DOINGが最初に視線が行く場所であること
- 他セクションの視認性が損なわれていないこと
